I need a feature in the Pos / Counter Balance section. Currently, I can only see the tax calculations there, but I cannot directly change or configure the tax percentages from that page.

Right now, tax settings can only be modified through the Admin Settings page, where taxes can be enabled, disabled, or configured.

What I want is a small dropdown (quick selector) inside the Counter Balance section itself. The dropdown should allow me to quickly select predefined tax percentages such as:

5%
10%
20%
Other configured tax rates

When I select a tax percentage from the dropdown, it should immediately apply to the calculation on that page without requiring me to navigate to the Admin Settings page.

The available tax percentages in this dropdown should be fetched from the Admin Settings configuration. The tax rate marked as the default in Admin Settings should be automatically pre-selected when the page loads.

In short:

Admin Settings remains the place where taxes are created, configured, enabled/disabled, and a default tax is chosen.
The Counter Balance section gets a quick tax-selection dropdown.
The dropdown shows only active/enabled tax rates.
The default tax from Admin Settings is automatically selected.
Users can switch between available tax rates directly from the Counter Balance page for faster calculations.
